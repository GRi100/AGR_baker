# AGR Baker v2.0

Аддон для Blender 5.0 для запекания и управления текстурными сетами с интеграцией Photoshop.

## 🎯 Основные возможности

- **Запекание текстурных сетов** с выбранных объектов на активный
- **Автоматическое создание всех необходимых текстур**:
  - T_MaterialName_Diffuse (RGB)
  - T_MaterialName_DiffuseOpacity (RGBA)
  - T_MaterialName_Emit (Emission Strength)
  - T_MaterialName_Roughness
  - T_MaterialName_Opacity (извлеченная прозрачность)
  - T_MaterialName_Normal (OpenGL/DirectX)
  - T_MaterialName_ERM (комбинированная карта Emit+Roughness+Metallic)
  - T_MaterialName_Metallic
- **Система текстурных сетов** (S_MaterialName) для организации текстур
- **Автоматическое подключение** текстур к материалам
- **Загрузка сетов** из папки AGR_BAKE
- **Интеграция с Photoshop** для обработки текстур
- **Модульная архитектура** для легкого расширения

## 📦 Установка

1. Скопируйте папку `AGR_baker_v2` в папку аддонов Blender:
   - Windows: `%APPDATA%\Blender Foundation\Blender\5.0\scripts\addons\`
   - Linux: `~/.config/blender/5.0/scripts/addons/`
   - macOS: `~/Library/Application Support/Blender/5.0/scripts/addons/`

2. Откройте Blender и перейдите в Edit → Preferences → Add-ons

3. Найдите "AGR Baker v2" и активируйте аддон

4. Панель появится в 3D Viewport → Sidebar (N) → AGR Baker

## 🚀 Быстрый старт

### Запекание текстур

1. **Подготовьте сцену**:
   - Выберите high-poly объекты (источники)
   - Выберите low-poly объект последним (он станет активным)
   - Убедитесь, что у активного объекта есть материалы

2. **Настройте параметры**:
   - Выберите разрешение текстур (512-4096)
   - Включите "Bake with Alpha" если нужна прозрачность
   - Выберите тип нормалей (OpenGL/DirectX)

3. **Запеките**:
   - Нажмите "Bake Texture Set"
   - Текстуры сохранятся в `AGR_BAKE/S_MaterialName/`

### Работа с текстурными сетами

1. **Обновить список сетов**:
   - Откройте панель "Texture Sets"
   - Нажмите "Refresh Sets"

2. **Подключить сет к материалу**:
   - Выберите сет из списка
   - Нажмите "Connect to Material" или "Assign to Active"

3. **Загрузить все сеты**:
   - Нажмите "Load All" для автоматического подключения всех сетов к соответствующим материалам

## 🔧 Настройки запекания

### Основные параметры

- **Resolution**: Разрешение текстур (512, 1024, 2048, 4096)
- **Bake with Alpha**: Запекать диффузную карту с альфа-каналом
- **Normal Type**: Тип карты нормалей (OpenGL или DirectX)
- **Max Ray Distance**: Максимальная дистанция луча для запекания
- **Extrusion**: Выдавливание клетки для запекания

### Режимы подключения текстур

Аддон автоматически определяет режим подключения:

- **HIGH режим**: Если есть ERM и DiffuseOpacity
  - Использует комбинированные карты для оптимизации
  
- **LOW режим**: Если используются отдельные карты
  - Подключает каждую текстуру отдельно

## 🐛 Решение проблем

### Аддон не появляется в списке

- Убедитесь, что папка называется `AGR_baker_v2`
- Проверьте, что все файлы на месте
- Перезапустите Blender

### Ошибка при запекании

- Сохраните blend файл перед запеканием
- Убедитесь, что у активного объекта есть материалы
- Проверьте, что выбраны mesh объекты

### Текстуры не подключаются

- Обновите список сетов кнопкой "Refresh Sets"
- Проверьте, что папка AGR_BAKE существует
- Убедитесь, что текстуры сохранены в правильном формате

## 📚 Дополнительная информация

### Требования

- Blender 5.0 или выше
- Python 3.11+
- NumPy (для обработки изображений)

### Производительность

- Используется CPU для запекания (стабильность)
- Samples = 1 для быстрого запекания
- Denoising отключен

### Поддерживаемые форматы

- Текстуры: PNG (8-bit)
- Colorspace: sRGB для цветных карт, Non-Color для данных
