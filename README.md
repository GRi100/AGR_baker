# AGR Baker v2

Aддон для **Blender 5.0+**, предназначенный для запекания текстур, создания текстурных атласов, работы с UDIM-тайлами и массового переименования ассетов.

**Автор:** computer_invader
**Версия:** 2.0.0
**Расположение в Blender:** View3D > Sidebar (N) > AGR Baker

---

## Возможности

### Запекание текстур
- **Selected-to-Active** — запекание с высокополигональной модели на низкополигональную с настройкой дистанции лучей и экструзии
- **Simple Bake** — запекание активного материала на плоскость
- **Batch Bake** — пакетное запекание всех материалов объекта
- Умное определение разрешения по нодовому дереву шейдера
- Поддержка CPU и GPU рендера, денойзинг

### Управление текстурными сетами
- Автоматическое сканирование папки `AGR_BAKE/` и построение списка сетов
- Подключение текстур к материалам (HIGH и обычный режим)
- Назначение материалов на активный объект
- Ресайз текстур (LANCZOS) — требуется Pillow
- Гауссово размытие — требуется Pillow
- Сортировка по имени, разрешению, наличию альфа-канала
- Выборка по критериям (альфа, разрешение, объект, материал)
- Пакетное удаление типов текстур (DiffuseOpacity, ERM, Normal)

### Текстурные атласы
- Упаковка текстурных сетов в атлас (алгоритм Guillotine)
- 2D-превью раскладки перед созданием
- Автоматический ремап UV-координат
- Применение атласа к объекту
- Распаковка атласа обратно в отдельные сеты

### UDIM-тайлинг
- Создание UDIM-набора из текстурных сетов (тайлы 1001, 1002...)
- Добавление сетов в существующий UDIM
- Разборка UDIM обратно в отдельные сеты
- Сохранение метаданных в `mapping.json`

### Конвертация материалов
- Извлечение текстур из нодового дерева материала в текстурные сеты
- Автоматическое разделение ERM-текстуры на каналы (Emission, Roughness, Metallic)
- Извлечение альфа-канала из DiffuseOpacity
- Конвертация одного или всех материалов объекта

### Переименование ассетов
- Объекты → `SM_Адрес[_Номер]_Тип` (Main, MainGlass, Ground, GroundGlass, GroundEl, GroundElGlass, Flora)
- Материалы → `M_Адрес[_Номер]_Тип_Индекс` / стёкла → `M_Glass_NN`
- Текстуры → `T_Адрес[_Номер]_Тип_ТипТекстуры_Индекс.png`
- Коллизии → `UCX_SM_Адрес[_Номер]_Тип_Индекс`
- GeoJSON → `SM_Адрес[_Номер].geojson` / FBX → `SM_Адрес[_Номер].fbx`
- Свет → родитель `SM_Адрес[_Номер]_Light`, дети `Light_Адрес[_Номер]`
- **Переименование всего проекта** одной кнопкой (объекты, материалы, текстуры, свет, GeoJSON, FBX)

### Рамки и бордюры
- Добавление рамок к текстурным сетам с настраиваемым масштабом
- Растекание краёв (edge bleed) через numpy
- Два варианта оверлея: сложный и минимальный
- Обработка произвольных PNG-файлов через файловый браузер
- Требуется Pillow

### Quick Mode
- Модальный оператор с HUD-оверлеем во вьюпорте
- Быстрый доступ ко всем основным функциям через клавиатуру

## Требования

| Компонент | Версия | Обязательность |
|-----------|--------|----------------|
| Blender | 5.0.0+ | Обязательно |
| Pillow (PIL) | любая | Опционально |

> **Pillow** необходим для ресайза текстур, создания рамок и конвертации материалов. Аддон автоматически определяет отсутствие Pillow и предлагает установку одной кнопкой прямо из интерфейса.

## Установка

1. Скопируйте папку `AGR_baker_v2/` в каталог аддонов Blender:
   - Windows: `%APPDATA%\Blender Foundation\Blender\5.0\scripts\addons\`
   - macOS: `~/Library/Application Support/Blender/5.0/scripts/addons/`
   - Linux: `~/.config/blender/5.0/scripts/addons/`
2. В Blender откройте **Edit → Preferences → Add-ons**
3. Найдите **AGR Baker v2** и включите его
4. Панель появится в боковой панели 3D-вьюпорта: **N → AGR Baker**

## Режимы текстур

Аддон поддерживает два режима организации текстур:

### HIGH — упакованные текстуры

| Файл | Содержимое |
|------|-----------|
| `T_Name_DiffuseOpacity.png` | RGBA — диффуз в RGB, прозрачность в альфа-канале |
| `T_Name_ERM.png` | RGB — R=Emission, G=Roughness, B=Metallic |
| `T_Name_Normal.png` | OpenGL карта нормалей |

### LOW — раздельные текстуры

| Файл | Содержимое |
|------|-----------|
| `T_Name_Diffuse.png` | RGB диффуз |
| `T_Name_DiffuseOpacity.png` | RGBA диффуз + альфа (опционально) |
| `T_Name_Roughness.png` | Шероховатость |
| `T_Name_Metallic.png` | Металличность |
| `T_Name_Opacity.png` | Прозрачность |
| `T_Name_Emit.png` | Эмиссия |
| `T_Name_Normal.png` | OpenGL карта нормалей |
| `T_Name_ERM.png` | Упакованная ERM (опционально) |

## Соглашения именования

> `[_Номер]` — опциональный 3-значный номер с ведущими нулями (001–999), используется для типов Main и MainGlass

### Объекты

| Шаблон | Пример |
|--------|--------|
| `SM_{Адрес}_{Тип}` | `SM_Broadway10_Ground` |
| `SM_{Адрес}_{Номер}_{Тип}` | `SM_Broadway10_001_Main` |

Blender может добавлять суффикс `.001` при дубликатах — он автоматически убирается при парсинге.

**Допустимые типы:** Main, MainGlass, Ground, GroundGlass, GroundEl, GroundElGlass, Flora

### Материалы

| Шаблон | Пример | Когда |
|--------|--------|-------|
| `M_{Адрес}_{Тип}_{Индекс}` | `M_Broadway10_Ground_1` | Объект без номера |
| `M_{Адрес}_{Номер}_{Тип}_{Индекс}` | `M_Broadway10_001_Main_1` | Объект с номером |
| `M_Glass_{NN}` | `M_Glass_01` | Стёкла (NN = 01–99) |

`{Индекс}` — порядковый номер слота материала (начиная с 1).

### Текстурные сеты и файлы

| Тип | Шаблон | Пример |
|-----|--------|--------|
| Папка сета | `S_{ИмяМатериала}` | `S_Wall` |
| Папка сета с рамкой | `S_{ИмяМатериала}_Frame` | `S_Wall_Frame` |
| Текстура (до переименования) | `T_{ИмяМатериала}_{ТипТекстуры}.png` | `T_Wall_Diffuse.png` |
| Текстура (после переименования) | `T_{Адрес}_{Номер}_{Тип}_{ТипТекстуры}_{Индекс}.png` | `T_Broadway10_001_Main_Diffuse_1.png` |
| Текстура Ground | `T_{Адрес}_Ground_{ТипТекстуры}_{Индекс}.png` | `T_Broadway10_Ground_Diffuse_1.png` |

**Типы текстур:** Diffuse, DiffuseOpacity, Roughness, Metallic, Opacity, Emit, Normal, ERM

### Атласы

| Тип | Шаблон | Пример |
|-----|--------|--------|
| Папка HIGH атласа | `A_{ИмяАтласа}` | `A_Broadway10_Main` |
| Текстура HIGH атласа | `T_{ИмяАтласа}_{ТипТекстуры}.png` | `T_Broadway10_Main_DiffuseOpacity.png` |
| Текстура LOW атласа | `T_{Адрес}_{Тип}_{суффикс}.png` | `T_Broadway10_Main_d.png` |

**Суффиксы LOW атласа:** `_d` (diffuse), `_do` (diffuseOpacity), `_r` (roughness), `_m` (metallic), `_o` (opacity), `_n` (normal), `_e` (emit), `_erm`

### UCX коллизии

| Шаблон | Пример |
|--------|--------|
| `UCX_SM_{Адрес}_{Тип}_{Индекс}` | `UCX_SM_Broadway10_Ground_001` |
| `UCX_SM_{Адрес}_{Номер}_{Тип}_{Индекс}` | `UCX_SM_Broadway10_001_Main_001` |

`{Индекс}` — 3-значный с ведущими нулями (001, 002...).

### Свет

| Тип | Шаблон | Пример |
|-----|--------|--------|
| Родитель (EMPTY) | `SM_{Адрес}_{Номер}_{Тип}_Light` | `SM_Broadway10_001_Main_Light` |
| Дочерние объекты | `Light_{Адрес}_{Номер}` | `Light_Broadway10_001` |
| Родитель Ground | `SM_{Адрес}_Ground_Light` | `SM_Broadway10_Ground_Light` |
| Дочерние Ground | `Light_{Адрес}_Ground` | `Light_Broadway10_Ground` |

### GeoJSON и FBX

| Тип | Шаблон | Пример |
|-----|--------|--------|
| GeoJSON (с номером) | `SM_{Адрес}_{Номер}.geojson` | `SM_Broadway10_001.geojson` |
| GeoJSON (Ground) | `SM_{Адрес}_Ground.geojson` | `SM_Broadway10_Ground.geojson` |
| FBX (с номером) | `SM_{Адрес}_{Номер}.fbx` | `SM_Broadway10_001.fbx` |
| FBX Light | `SM_{Адрес}_{Номер}_Light.fbx` | `SM_Broadway10_001_Light.fbx` |

### UDIM

| Тип | Шаблон | Пример |
|-----|--------|--------|
| Папка Main | `SM_{Адрес}` | `SM_Broadway10` |
| Папка Ground | `SM_{Адрес}_Ground` | `SM_Broadway10_Ground` |
| Текстура | `T_{Адрес}_{ТипТекстуры}_{Индекс}.{UDIM}.png` | `T_Broadway10_001_Diffuse_1.1001.png` |

UDIM-тайлы нумеруются начиная с 1001. Метаданные хранятся в `udim_mapping.json`.

### Lowpoly-коллекции

Коллекции, начинающиеся с 4 цифр (например `0903`, `2024`), распознаются как lowpoly-коллекции при переименовании проекта.

## Quick Mode — клавиатурные сочетания

Активация: кнопка **Quick Mode** на главной панели

| Клавиша | Действие |
|---------|----------|
| `Q` | Запечь с высокополигональной модели |
| `Q + Alt` | Запечь с нормалями |
| `Q + Ctrl` | Запечь с альфа-каналом |
| `Q + Alt + Ctrl` | Запечь с нормалями и альфой |
| `E` | Конвертировать активный материал |
| `E + Shift` | Конвертировать все материалы |
| `R` | Простое запекание активного материала |
| `R + Shift` | Простое запекание всех материалов |
| `W / S` | Дистанция лучей +/- 0.01 |
| `A / D` | Экструзия +/- 0.01 |
| `Колесо мыши` | Изменение разрешения (64–4096) |
| `ESC / ПКМ / ЛКМ` | Выход из Quick Mode |
